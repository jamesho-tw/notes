# KALI

## Change locale
```
$> dpkg-reconfigure locales
```

## Install Traditional Chinese font
```
apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-wqy
```

## Install Traditional Chinese input method
```
$> sudo apt-get install scim-chewing
```
