# Syntax highlighting
* Turn on
```
:syntax on
```
* Turn off
```
:syntax off
```
* Enable vim syntax highlighting by default  
_it can be found in your home directory, if you've not found and create it._
```
$> vi ~/.vimrc

syntax on
```


#	Display line numbers
* Enabling
```
:set nu
```
* Disabling
```
:set nu!
```


# Turn off the backup and swap file
```
$> vi ~/.vimrc

set nobackup
set nowritebackup
set noswapfile
```
